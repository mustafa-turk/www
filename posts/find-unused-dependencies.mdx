---
title: Find unused dependencies and modules in your project
date: 12/04/2022
---

If you are working on a large Javascript project, chances are big that there are many left over components that are not being used anymore. Dangling files complicate the codebase for no obvious reason. If you don't use the file or dependency anymore, then why keep it around?

Using the following command, you can list all the unused files and dependencies in your project:

```shell
npx unimported
```
